# KRYSTALION HQ — Luxury Hotel Website

A fully responsive ultra-luxury hotel website built with pure HTML, CSS and JavaScript.


## ✨ Features
- Ultra-luxury design (black & gold theme)
- Standard rooms from ₦25,000/night
- Luxury apartments up to ₦1,000,000/night (Presidential Suite)
- Interactive booking form with price calculator
- Sliding image galleries for each suite
- Full dining menu modal
- Hotel services accordion
- Staycation enquiry form
- Newsletter subscription
- Fully mobile responsive
- Smooth scroll animations

## 🛠 Built With
- HTML5
- CSS3 (15 separate stylesheets)
- Vanilla JavaScript (8 separate scripts)
- Google Fonts (Cinzel, Cormorant Garamond, Montserrat)
